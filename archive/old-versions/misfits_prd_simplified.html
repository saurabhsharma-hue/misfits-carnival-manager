  • Geographic clustering when possible
  • New clubs added only when slot opens

  • üìâ Leading Indicators: >80% failure confidence

Key Algorithms

Health Calculation

health = (leading_indicators * 0.6) + (current_metrics * 0.4)
where:
- leading_indicators = avg(booking_velocity, feedback_sentiment, leader_engagement)
- current_metrics = avg(revenue_trend, capacity, repeat_rate)

Pattern Confidence

confidence = (historical_accuracy * 0.5) + (signal_strength * 0.3) + (sample_size * 0.2)
where:
- historical_accuracy = times_correct / total_predictions
- signal_strength = correlation_coefficient
- sample_size = min(1, affected_clubs / 10)
